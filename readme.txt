Connect Homey Flows to Frigate MQTT events. This app listens for tracked object metadata updates, review messages, doorbell presses, and state classification changes, then exposes trigger cards for object descriptions, face recognition, license plates, review starts, alert escalation, review endings, GenAI review summaries, and doorbell activity. It also adds a Frigate Camera device with snapshot and live view support.

Configure your MQTT broker and Frigate topic prefix in app settings. A Frigate base URL can also be set so thumbnail paths are returned as full URLs in Flow tokens. This app was coded almost entirely with AI assistance and reviewed lightly. The architectural guidance was provided by a developer with more than 30 years of experience.
