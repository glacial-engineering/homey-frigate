Connect Homey Flows to Frigate MQTT events. This app listens for tracked object metadata updates and review messages, then exposes trigger cards for object descriptions, face recognition, license plates, review starts, alert escalation, review endings, and GenAI review summaries.

Configure your MQTT broker and Frigate topic prefix in app settings. A Frigate base URL can also be set so thumbnail paths are returned as full URLs in Flow tokens.
